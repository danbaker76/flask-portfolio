UFC Fight Predictor

Enter two fighter names exactly as they appear in the UFC Stats database (e.g., "Jon Jones" vs "Stipe Miocic"). The machine learning model (XGBoost) estimates the win probability for the red corner.

⏳ Waking up the server (cold start)… this may take up to 40 seconds.
