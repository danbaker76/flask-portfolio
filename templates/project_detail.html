{{ project.title if project else 'Real-time Crypto Pipeline' }}

✅ Production ready 2025

Problem

Financial analysts needed sub‑second latency on crypto prices to detect arbitrage opportunities. Existing batch pipelines were too slow.

Solution

A streaming pipeline that consumes from Coinbase WebSocket, cleans with Spark Structured Streaming, and writes to Parquet on S3.

Architecture

            graph LR
                A[Coinbase WebSocket] --> B[Kafka]
                B --> C[Spark Streaming]
                C --> D[Raw S3 Bucket]
                D --> E[Glue Catalog]
                E --> F[Athena / Redshift]
